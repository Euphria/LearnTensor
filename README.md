# Create Environment
因为包括且不限于贫穷、自然灾害、猫娘骚扰，我没有办法拥有一个集群，所以我只能用我的笔记本去写点小东西。很显然，我的笔记本是**Win11**

所以你可以用下面的指令创建一个conda环境

```bash
cd LearnTensor
conda env create -f conda_env.yml
```
或者一边运行等报错，一边进行一个python环境的构建（毕竟pytorch版本因机而异）
